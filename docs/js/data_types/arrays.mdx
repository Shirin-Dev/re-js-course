Мы уже знакомы с объектами и знаем, а давайте рассмотрим такой объект:

```js
let pseudo_array = {
    0: "a",
    1: "b",
    2: "c",
};
```

Перед нами удобный способ хранить последовательность значений, каждому значению в массиве присвоен номер(**индекс**).

## Методы массивов

### `Array.prototype.push()`

Этот метод позволяет нам добавить новый элемент в массив.

### `Array.prototype.splice(start, [deleteCount], [items...])`

Этот метод позволяет нам заменить ряд элементов в массиве или добавить новый элемент в произвольное место.

#### Аргументы

-   `start` - с какой позиции мы начинаем заменять элементы
-   `deleteCount` - сколько элементов с этой позиции надо удалить
-   `items` - какие элементы нам надо добавить с указанной позиции
