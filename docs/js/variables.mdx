Давайте для начала представим, что переменные это коробочки с именами.

В коробочки мы можем ложить самые разные значения, переписывать эти значения.

Переменные объявляются с помощью оператора `var`

_Существуют операторы `let` и `const`, но их изучение будет позже. Эти операторы появились в языке относительно недавно._

```jsx
var foo; // объявили переменную
foo = 1; // присвоили переменной значение
foo = 2; // переопределили значение переменной
console.log(bar); // попробовали обратиться к необъявленной переменной... это приведет к ошибке
var baz = 1; // объявили переменную и присвоили ей значение
```

## Имена переменных

На имя переменной в JavaScript наложены всего три ограничения.

1. Имя может состоять из: букв, цифр, символов `$` и `_`
2. Первый символ не должен быть цифрой.
3. Имя не может быть таким же как одно из ключевых слов(операторы)
   `break`, `case`, `catch`, `continue`, `debugger`, `default`, `delete`, `do`, `else`, `false`, `finally`, `for`, `function`, `if`, `in`, `instanceof`, `new`, `null`, `return`, `switch`, `this`, `throw`, `true`, `try`, `typeof`, `var`, `void`, `while`, `with`

\*Если вы переопределите одну из глобальных переменных своей переменной тоже ничего хорошего не будет. **Это очень плохо!\***

```jsx
var break = 1; // ошибка!
var alert = 2;
alert(); // ошибка! теперь у нас нет такой функции alert!
```

## Всплытие переменных(hoisting)

_Один из неприятных моментов._

Объявление переменных (как и любые другие объявления) обрабатываются до выполнения кода. Где бы не находилось объявление, это равнозначно тому, что переменную объявили в самом начале кода. Это значит, что переменная становится доступной до того, как она объявлена. Такое поведение называется "**всплытием**"

```jsx
console.log(foo); // выведет в консоль undefined
var foo = 1;
```

_Принято не использовать переменные в коде, если они еще не объявлены. А вообще следует объявлять переменные в начале функции или файла._

_В новом стандарте. Если переменная объявлена через `let` или `const` всплытие не произойдет_

## Константы

Выше я уже упомянул оператор `const` но сейчас речь не о нем.

**Константа** – это переменная, которая никогда не меняется. Как правило, их называют большими буквами, через подчёркивание. Например:

```jsx
var COLOR_RED = "#F00";
var COLOR_GREEN = "#0F0";
var COLOR_BLUE = "#00F";
var COLOR_ORANGE = "#FF7F00";
```

Технически... мы можем переопределить эти переменные, но есть соглашение между разработчиками, что так делать нельзя
